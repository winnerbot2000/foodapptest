# Support

## Getting Help

- Use GitHub Issues for reproducible bugs and actionable feature ideas.
- Use the provided issue templates so reports are easier to triage.
- For general repository questions, open an issue and label the context clearly in the title.

## Security

For security-sensitive issues, follow [SECURITY.md](SECURITY.md) and avoid posting details publicly.

## Good Support Requests

Helpful requests usually include:

- What you expected to happen
- What actually happened
- Device or simulator details
- Xcode version
- Steps to reproduce
- Screenshots if the issue is visual
