# Code of Conduct

## Our Commitment

We want this project to be a respectful, welcoming place for contributors of all experience levels. Participation should feel safe, constructive, and focused on making the app better together.

## Expected Behavior

- Be respectful and assume good intent
- Give feedback about the work, not the person
- Be open to questions, learning, and different levels of experience
- Share context clearly when reporting bugs or suggesting changes
- Resolve disagreements with curiosity and professionalism

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or deliberately disruptive behavior
- Publishing private information without permission
- Repeatedly derailing discussions away from the project

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, pull requests, or other contributions that do not align with this code of conduct.

## Scope

This code of conduct applies in repository discussions, issues, pull requests, and other project communication spaces.

## Reporting

For conduct concerns, contact the maintainer privately through GitHub rather than escalating the issue in public threads.
